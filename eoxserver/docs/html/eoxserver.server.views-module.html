ows(request)

source code 
This function handles all incoming OWS requests. It configures basic settings e.g. for the logging module, triggers reading of the config file and passes on the request to eoxserver.lib.ows.EOxSOWSHandler.
Parameters:
  • request - A django.http.HttpRequest object containing the request parameters and data
Returns:
A django.http.HttpResponse object containing the response content, headers and status

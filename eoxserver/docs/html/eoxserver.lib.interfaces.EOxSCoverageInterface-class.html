getDatasets(self, **kwargs)

source code 
Returns the EO datasets contained in this coverage; if this coverage is a dataself itself, returns a reference to this coverage interface. If provided, the filenames are selected according to the keyword arguments included in the request; the specific behaviour depends on the type of coverage.
Parameters:
  • kwargs - keyword arguments for selection of datasets
Returns:

addLayers(self, ms_req)

source code 
This method adds layers to the ms_req.map object based on the coverages defined in ms_req.coverages. The default is to unconditionally add a single layer for each coverage defined. This method can be overridden in order to customize the way layers are inserted into the map object.

getMapServerLayer(self, coverage, **kwargs)

source code 
This method is invoked by the addLayers method in order to generate mapscript.layerObj instances for each coverage. The basic configuration is done here, but subclasses will have to override this method in order to define e.g. the data sources for the layer.
Parameters:
  • coverage - An eoxserver.lib.interfaces.EOxSCoverageInterface object giving access to the coverage data
Returns:

OGC Georeferenced Table Joining Service Implementation Standard
-----------------------------------------------------------------------

The OpenGIS(r) Georeferenced Table Joining Service Implementation Standard (TJS) 
is the OGC standard which defines a simple way to describe and exchange tabular 
data that contains information about geographic objects.

More information may be found at
 http://www.opengeospatial.org/standards/tjs

The most current schema are available at http://schemas.opengis.net/ .
